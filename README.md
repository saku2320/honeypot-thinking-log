# Honeypot Thinking Log

ハニーポット環境を作るまでに考えたことを置いていくリポジトリ。

まだ構築作業には入ってないし、具体的な設定も固まってないので、
ここにある内容は完成済みの手順書ではなく、ほぼ頭の中のメモです。

「こうしたらよさそう」
「いや、これ危なくね？」
「じゃあこうした方がいいかも」

みたいな思考の流れを、できるだけそのまま残す予定です。

かなりラフに文字起こししていくので、普通の技術記事というよりはかなり独り言に近いです。
喋り口調や適当なノリもそのまま残す予定なので、そのあたりはご了承ください。


ただし、独り言メインと言っても公開リポジトリなので例として以下の情報や載せるべきでないと判断した内容については載せません。（そりゃそうだろって項目ばかりではありますが...）
- 実際に使用しているグローバルIP
- 内部IPやVLAN IDなど、環境を特定できる情報
- 学校名・関係者名
- パスワード・鍵・管理URL
- まだ塞いでいない脆弱性の具体情報
